"Первый HTTP сервер"
1
— Реализуйте HTTP хендлер, который в ответе возвращает какую-нибудь одну случайную заранее заготовленную фразу
— Запустите HTTP сервер и протестируйте написанный HTTP хендлер
2
— Напишите 3 HTTP хендлера:
1. Первый хендлер будет возвращать "Я собака и я говорю 'Гав'"
2. Второй хендлер будет возвращать "Я кошка и я говрою 'Мяу'"
3. Третиь хендлер будет возвращать "Я корова и я говорю 'Мууу'"
— Каждый хендлер должен обрабатываться на своём уникальном паттерне
— Запустите HTTP сервер и протестируйте написанные хендлеры
