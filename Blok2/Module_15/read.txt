"Тело HTTP запроса"
1
— Напишите программу, которая предоставляет возможность принимать пришедшие по сети текстовые сообщения и сохранять их внутри себя
— Мы, как клиенты, можем отправлять в эту программу на определённый эндпоинт свои сообщения по HTTP, а программа сохранит их и выведет на экран
— Необходимо протестировать написанную программу
2
— Мы модифицируем программу из предыдущего задания
— Отныне каждое пришедшее сообщение маркируется уникальным случайно-сгенерированным ID'шников
— Наша программа, помимо возможности сохранять сообщения, теперь предоставляет возможость удалять сообщения по их ID
— То есть линия взаимодействия клиента с программой такова:
1. Клиент отправляет по HTTP нам несколько сообщений, текст сообщения передаётся в теле запроса. Мы каждому пришедшему сообщению генерируем случайный ID, после чего сохраняем.
2. Клиент может отправить HTTP запрос на удаления сообщения, ID удаляемого сообщения передаётся в теле запроса.
3. Каждый раз мы выводим информацию о том, какое сообщение добавлено или удалено, а так же информацию о том, какие вообще сообщения у нас сохранены, в консоль.
