 "Тело HTTP ответа и статус коды"
 1
 — Опишите HTTP хендлер, который в зависимости от цифры, переданной в HTTP запросе, будет отдавать различные HTTP ответы и различные статус-коды
 2
 — Мы модифицируем программу из второго домашнего задания 15-й темы
 — Помимо добавления и удаления сообщений, необоходимо реализовать хендлер, на который можно сделать запрос, и в HTTP ответе получить список всех имеющихся на данных момент сообщений
 — Также необходимо реализовать хендлер, который в теле запроса принимает ID сообщения, а в теле ответа отдаёт текст сообщения, которому принадлежит переданный ID. Если сообщения с переданным ID нет, в HTTP тело ответа необходимо положить информацию об ошибке, в HTTP статус код необходимо положить 404 Not Found
