 "HTTP методы"
 1
 — Опишите любой простой хендлер, который бы обрабатывал запросы только на определённом HTTP методе (метод выберите самостоятельно, выбор случаен)
 — Если клиент делает запрос с неверно выбранным HTTP методом, необходимо вывести к консоль сообщение об ошибке, а статус код должен быть 405 Method Not Allowed.
 — Если клиент делает запрос с верно выбранным HTTP методом, то необходимо вывести в консоль информацию о том, что HTTP хендлер отработал успешно.
 2
 — Мы модифицируем программу из второго домашнего задания 16-й темы
 — Каждому хендлеру необходимо задать HTTP метод, который этот хендлер будет обрабатывать:
 1. Хендлер для сохранения новых сообщений должен иметь HTTP метод "POST"
 2. Хендлер для удаления уже имеющихся сообщения должен иметь HTTP метод "DELETE"
 3. Хендлер для получения списка всех имеющихся сообщения должен иметь HTTP метод "GET"
 — В случае, если HTTP клиент вызывает хендлер, выбирая неверный HTTP метод, в теле ответа необходимо вернуть сообщение об ошибке, а статус код в ответе должен быть 405 Method Not Allowed
